Synthesize multiple remembered facts into a single observation.

You have access to the context-fabrica MCP server tools.

$ARGUMENTS

If record IDs were provided, use the `synthesize` tool directly with those IDs.

If a topic was provided instead of IDs, first use the `recall` tool to find relevant records about that topic, then use `synthesize` on the most relevant record IDs to create a provenance-backed observation.

After synthesizing, show the new observation including what records it was derived from.
